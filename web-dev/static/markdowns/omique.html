Omique 消息队列框架

作者：stormi-li

Email：2785782829@qq.com

简介

本地消息队列框架，消息点对点传递，不通过中间件。

功能

  • 支持生产和消费
  • 支持接入服务注册与发现系统

教程

安装

go get github.com/stormi-li/omique-v1

创建生产者

package main

import (
	"fmt"

	"github.com/go-redis/redis/v8"
	omique "github.com/stormi-li/omique-v1"
)

func main() {
	c := omique.NewClient(&redis.Options{Addr: "localhost:6379"})
	producer := c.NewProducer("queue")
	err := producer.Publish([]byte("hello world"))
	if err != nil {
		fmt.Println(err)
	}
}

创建消费者

package main

import (
	"fmt"

	"github.com/go-redis/redis/v8"
	omique "github.com/stormi-li/omique-v1"
)

func main() {
	c := omique.NewClient(&redis.Options{Addr: "localhost:6379"})
	consumer := c.NewConsumer("queue", "localhost:5555")
	consumer.ListenAndConsume(10000, 1, func(message []byte) {
		fmt.Println(string(message))
	}) //参数为队列容量和权重
}
